# Farhan Aulianda - Interactive Portfolio

Welcome to the source repository for my interactive portfolio. This repository demonstrates my technical capabilities, software engineering practices, and system architecture design.

## Live Access

The portfolio is deployed and accessible across two distinct interfaces. You are encouraged to explore both environments to get a comprehensive view of my work.

### Web Interface
A modern, responsive web application showcasing my projects and professional experience.
- **URL:** [https://farhanaulianda.my.id](https://farhanaulianda.my.id)

### Terminal Interface
A custom SSH server providing a command-line experience. This interface allows you to navigate my portfolio directly from your terminal.
- **Command:** `ssh ssh.farhanaulianda.my.id`

## Repository Structure

This is a monorepo containing the following applications:

- **`apps/web`**: The frontend web application built with Svelte.
- **`apps/ssh`**: The custom backend SSH server written in Go, powering the terminal interface.

## Architecture & Principles

The development of this portfolio adheres to several core principles:
- **Simplicity:** Clean, readable, and maintainable codebase.
- **Performance:** Optimized delivery for both the web frontend and terminal backend.
- **Professionalism:** Focus on user experience, stability, and secure design patterns.
